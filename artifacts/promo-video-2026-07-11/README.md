# 시험암기 홍보 영상

## 최종 결과물

- `exam-memorizer-promo-vertical-30s.mp4`
- 1080 × 1920, 세로 9:16
- 30.006초
- H.264 MP4 + 오디오 트랙 1개
- 음악과 UI 효과음은 렌더러가 직접 합성한 원본 음원

## 구성

1. 외울 내용이 쌓이는 문제 제시
2. 시험암기 브랜드와 핵심 문장 소개
3. 문장을 입력하고 가릴 답을 선택하는 흐름
4. 정답을 가리고 필요할 때 확인하는 흐름
5. `외웠어요` / `다시 불러요` 판단
6. PC와 아이폰의 같은 암기장 공유
7. 서비스 주소와 시작 CTA

## 파일

- `promo.html`: 30초 캔버스 애니메이션과 오디오 합성
- `render_promo.mjs`: 로컬 Chrome 기반 MP4 렌더러
- `preview-01.png` ~ `preview-06.png`: 장면별 대표 프레임

## 다시 렌더링

프로젝트 루트에서 실행합니다.

```bash
node artifacts/promo-video-2026-07-11/render_promo.mjs
```

렌더러는 기존 UX 감사 캡처를 읽기 전용 소스로 사용하며 앱 코드는 변경하지 않습니다.
