# 시험암기 홍보 영상 v2

- `exam-memorizer-cloze-promo-vertical-25s.mp4`
- 1080 × 1920, 세로 9:16
- 약 25초
- H.264 MP4 + 자체 합성 오디오

## 핵심 메시지

> 플래시카드와는 다르게, 여러 개는 함께 외우고 답은 하나씩 확인.

일반 플래시카드가 답 여러 개를 한 번에 공개하는 장면과, 시험암기가 같은 문맥의 빈칸 세 개를 하나씩 공개하는 장면을 직접 대비합니다. PC와 기기 동기화는 제외하고 모바일 학습 경험만 다룹니다.

## 타임라인

1. `플래시카드와는 다르게` 문제 제기
2. 일반 플래시카드에서 세 답이 한꺼번에 공개
3. 시험암기의 세 빈칸 시작 상태
4. 첫 번째 답만 확인하고 두 빈칸 유지
5. 두 번째 답까지 확인하고 마지막 빈칸 유지
6. `여러 개는 함께 외우고, 답은 하나씩 확인` 정리
7. 모바일 CTA
