시험암기 · 첫 사용 흐름 변경 비교

사용법을 추측하던 시작 화면을, 순서대로 따라가는 흐름으로 바꿨습니다.

이 페이지는 앱에 추가한 첫 사용 UX 개선을 검토하기 위한 비교 보고서입니다. 실제 암기 기능과 데이터 구조는 건드리지 않고, 사용자가 처음부터 끝까지 망설이는 지점을 정리했습니다.
